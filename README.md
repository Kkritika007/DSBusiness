# DS Lumping Services Website

A static website built for **DS Lumping Services**, a local business.
The site provides basic business information and an online presence using simple, fast-loading web technologies.

## 🌐 Overview

This project is a responsive static website created with **HTML and CSS**.
It was designed to showcase the services offered by DS Lumping Services and provide customers with an easy way to learn about the business.

## 🚀 Features

* Clean and simple design
* Responsive layout
* Fast-loading static pages
* Service information for customers
* Contact and business details

## 🛠️ Built With

* **HTML5**
* **CSS3**

## 📂 Project Structure

```
ds-lumping-services/
│
├── index.html
├── style.css
└── assets/
    └── images/
```

## 💻 How to Run

1. Clone the repository:

```
git clone https://github.com/yourusername/ds-lumping-services.git
```

2. Open the project folder.

3. Double-click **index.html** or open it in your browser.
